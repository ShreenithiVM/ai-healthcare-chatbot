# AI Healthcare Appointment & Chatbot System

## Tech Stack
- Django REST Framework (Backend)
- React (Frontend)
- PostgreSQL / SQLite
- Botpress (AI Chatbot Platform)
- RAG-based AI Integration

## Features
- Appointment booking and management system
- Scalable REST API backend
- AI-powered chatbot using Botpress
- Integration with RAG-based knowledge retrieval
- Fullstack architecture with backend and frontend components

## Note
The chatbot functionality is implemented using Botpress and integrated with a RAG-based system for enhanced, domain-specific responses. Certain AI components were developed collaboratively.
